Angular SuperGallery
===

Under construction

###Requirements
- AngularJS 1.6.4
- Bootstrap 3.3.7
- FBAngular (angular-fullscreen.js)

### Usage



### Build
- `npm install`
- `typings install`
- `gulp dev` or `gulp prod` for minified javascript and css file
